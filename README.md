# 🎟️ Ticket Booking System

A modern and responsive ticket booking web application built using HTML, CSS, JavaScript, and Tailwind CSS. Users can book tickets easily and apply coupon codes to receive discounts.

---

## 🚀 Features

* 🎫 Book tickets with an interactive UI
* 🔢 Maximum 4 tickets per user (booking limit)
* 💸 Apply coupon codes for discounts
* ❌ Invalid coupon handling system
* 📱 Fully responsive design
* ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Tailwind CSS
* 
## 🎯 Booking Rules

* A user can book **maximum 4 tickets at a time**
* Coupon codes can be applied for discounts
* Discount is applied instantly after validation

## 💡 Coupon System

* Users can enter coupon codes
* Valid codes provide discounts
* Invalid codes show error message
